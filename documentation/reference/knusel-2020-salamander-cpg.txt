MATERIALS AND METHODS

CPG Model

The model was developed using the Codyn framework and
exported to C code to run on the robot microcontrollers. Only
the 25 most rostral axial segments (each comprising 1 left and
1 right hemisegments) are modeled out of the 40 segments
that salamanders typically have (for review see, Chevallier et al.,
2008). Four additional oscillators control the limbs. The 25 axial
segments control the active part of the robot. The caudal half
of the robot tail is a passive, flexible caudal fin (Figure 1A).
Each axial hemisegmental oscillator and each limb oscillator is
modeled as a phase oscillator with controllable amplitude, and
the connections between oscillators are functions of the phase
difference between sender and receiver:
